## Project Title
CodeOrbitTech Landing Page

## Description
This project is a responsive landing page developed for CodeOrbit Tech. It showcases the company's services, mission, and call-to-action sections with a modern, user-friendly design.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Responsive design
- Modern user interface
- Hero section
- About section
- Services section
- Contact section
- Smooth navigation

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/naya-na/CodeOrbitTech_landingpage.git

2. Open the project folder.

3. Open the `index.html` file in your web browser.
